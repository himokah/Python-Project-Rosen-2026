# Python-Project-Rosen-2026
Doing the project did come with some difficulties. First problem was figuring out how exactly to start we didn't know anything about Pillow at all so we had to watch a couple of Youtube videos on what exactly we're working with. Second problem we considered is that it might not work for every image type. Through research, we learned that there different channels or colors depending on the image type. A PNG image was our worse nightmare as it had 4 different channels but we worked around it made it so that each image iis converted only to RGB rather than RGBA. Finally another issue we faced is how long it took to load. At first we had it where turtle would draw it on screen but it was painfull watching it trying to load any dot of the image. However, our trusty friend, the internet(stackoverflow) had a post on how to speed it up. Overall project wasn't too bad but it kinda beat our expectations on how long/difficult it was gonna be.
